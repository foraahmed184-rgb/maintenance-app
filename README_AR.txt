نسخة Firebase Realtime Firestore

هذه النسخة تجعل الطلبات تظهر للجميع من أي جهاز.

طريقة الرفع:
1. فك الضغط.
2. ارفع الملفات الأربعة فقط إلى GitHub:
   index.html
   app.js
   styles.css
   manifest.json
3. Commit changes.
4. انتظر دقيقة وافتح الرابط مع ?v=20

بيانات الدخول:
- عضو فريق: اكتب الاسم فقط واختر عضو من الفريق.
- عامل الصيانة: اختر عامل الصيانة وكلمة المرور 0000.
- المسؤول: اختر المسؤول وكلمة المرور 1234.

المميزات:
- كل الطلبات تظهر للجميع مباشرة.
- المسؤول يقدر يحذف.
- العامل والمسؤول فقط يغيرون الحالة.
- العضو يقدر يرسل تذكير.
- صور المشكلة وصور الإنجاز تفتح بحجم كبير.
